# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning techniques. The model is trained using passenger information such as age, gender, ticket class, fare, and family details.

The goal of this project is to understand data preprocessing, feature engineering, and classification modeling.

---

## 📂 Dataset
The dataset contains passenger details including:

- Passenger Class (Pclass)
- Sex
- Age
- Number of Siblings/Spouses (SibSp)
- Number of Parents/Children (Parch)
- Fare
- Embarked Location
- Survival Status

Target Variable:
- **Survived**
  - 0 → Did not survive
  - 1 → Survived
